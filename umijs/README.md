# 初始化流程

pwd：~/repos/frontend-scaffolding/umijs

## 创建项目

```shell
yarn create @umijs/umi-app
```

## 安装依赖

```shell
yarn install
```

## 启动项目

```shell
yarn start
#或
yarn run start
#或
yarn run umi dev
#或
./node_modules/.bin/umi dev
```

访问http://localhost:8000
